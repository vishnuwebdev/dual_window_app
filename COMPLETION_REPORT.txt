                          SUPPORT & RESOURCES
================================================================================

IF YOU GET STUCK:

1. Check the error message in the app UI
2. Look at console output (shows emoji progress)
3. Read SETUP_GUIDE.md → Troubleshooting section
4. Verify protocol buffer files copied correctly
5. Check network connectivity to locker backend

COMMON ISSUES:

"Cannot import service.pb.dart"
  → Copy protocol buffer files from cv-wep-frontend

"Backend showing offline"
  → Check locker IP address in Configuration
  → Verify backend hardware is powered on
  → Run: ping 192.168.1.100

"App won't start"
  → Run: flutter clean && flutter pub get
  → Check pubspec.yaml has all dependencies
  → Run with verbose: flutter run -v

"Door won't open"
  → Check backend service is running
  → Verify gRPC port (50051) is accessible
  → Check compartment ID is 1-12
  → Check physical hardware connections
